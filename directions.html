Steps to complete the riddle:

  1. To gather the letters for the riddle, you need to solve the questions in each of the tabs above, and write the letter of the corresponding answer down.
  2. Go to the proofs page, and complete the problems.
  3. Use the information on the proofs page to create constructions, where the directions are on the constructions tab.
  4. Go to the puzzles page and, using the information gathered from step 2, solve the puzzles.
  5. After completing the puzzles, use the answers that you have gotten for each one to answer the riddle. Type the letters that were gathered and input them into the text box provided.
